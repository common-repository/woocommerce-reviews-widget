WooCommerce widget automatically displays associated reviews for products on WooCommerce shop pages. 

== Description ==

Easily display needed product reviews associated with category and other shop pages with "WooCommerce Review Widget”

<h3>Features</h3>
<ul>
<li>Automatically displays associated reviews for products</li>
<li>Number of comments to show</li>
<li>Order by name, date, random, asc, desc</li>
<li>Product thumbnails on/off</li>
</ul>

<a href="http://wordpress.org/plugins/woocommerce/">WooCommerce</a> plugin is required

== Installation ==


1. Upload `woocommerce-reviews-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Appearance > Widgets 
4. Find "WooCommerce Reviews" and place it in sidebar
